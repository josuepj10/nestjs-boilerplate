# Teslo API

1. Clone repository

2. Execute
```
yarn install
```
3. Clone and configure .env.template


4. Deploy DB
```
docker-compose up -d
```

5. Excute
```
yarn start:dev
```

6. Excute seed
```
http://localhost:3000/api/seed
```

7. OpenApi (Swagger)
```
http://localhost:3000/api
```
